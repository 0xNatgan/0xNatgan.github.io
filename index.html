DocGen LLM
Internship Maintaining

A CLI tool that auto-generates high-quality documentation for any codebase. Uses Language Server Protocol servers for precise symbol and structure extraction, then passes that context to an LLM to produce summaries, docstrings, and Markdown docs. Language-agnostic by design — if an LSP exists for it, DocGen can document it. A Tcl LSP is available at this link: github.com/0xNatgan/tcl-lsp
