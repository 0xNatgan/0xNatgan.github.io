DocGen_LLM

Outil de génération de documentation de code à l'aide de modèles de langage et de RAG.
  • Python
  • Docker
  • Serveurs de langage
  • LLM
  • RAG
  • documentation
  • mkdocs
  • Markdown
  • HTML
  • open source
  • IA

Problématique principale : Générer une documentation de code de manière automatisée et agnostique au langage dans le cadre de projet manquants de documentation et/ou ayant une documentation peu claire

DocGen_LLM est un outil de génération de documentation de code qui utilise des modèles de langage en lien avec des serveurs de langage pour générer une documentation claire et concise de l'ensemble d'une base de code d'un projet. Il est conçu pour être agnostique par rapport aux langages de programmation. L'outil s'intègre facilement dans les outils de lecture de documentation tels que mkdocs. L'outil permet de générer une documentation à ajouter au code ou à utiliser dans une documentation externe utilisant markdown ou HTML

Caractéristiques :
  • Technologies utilisées : Python, Docker, Serveurs de langage, LLM, MKDocs
  • Objectifs : Automatiser la création de documentation de projet de code de manière agnostique au langage sur des projets manquants de documentation
  • Résultats : Amélioration de la lisibilité et de la compréhension du code, réduction du temps consacré à la documentation manuelle et augmentation de la qualité de la documentation générée
